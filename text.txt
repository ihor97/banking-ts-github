npm init - створюємо package.json
npm i typescript --save-dev встановлюємо typescript
npx - утіліта яка дозволяє запускати таски в простішому варіанті
npx tsc ./src/app.ts компілюємо 
npx tsc --init конфіг тайпскріпта

sourceMap - зберігається копія тайпскріпт файлів щоб можна було потім дебажити в браузері

dependencies -треба для продакшина

devDependencies - треба коли ми розробляємо

--save-dev -встановлюємо локально


"src/**/*" вказує на включення всіх файлів, що знаходяться в директорії src та всіх її піддиректоріях. Тобто директорія з якої будуть компілитися наші ts файли

**/: Ця частина шаблону вказує на будь-яку кількість піддиректорій. Знак ** представляє будь-яку кількість директорій, навіть нульову кількість (тобто безпосередньо в директорії src без піддиректорій).

*: Ця частина шаблону вказує на будь-який файл у кожній з піддиректорій. Знак * представляє будь-яку назву файлу.

"module":commonjs - вказує під яким модулем буле зібраний весь проект

commonjs- для ноди

npm i rimraf - для очистки директорій src

  "scripts": {
    "clear":"rimraf ./dist",
    "build":"npm run clear&&tsc"
  },
